lingvo.tasks.asr.decoder_utils.ComputeWer(hyps, refs)[source]¶

Computes word errors in hypotheses relative to reference transcripts.

Parameters:
  • hyps – Hypotheses, represented as string tensors of shape [N].
  • refs – References, represented as string tensors of shape [N].
Returns:

An int64 tensor, word_errs, of size [N, 2] where word_errs[i, 0] corresponds to the number of word errors in hyps[i] relative to refs[i]; word_errs[i, 1] corresponds to the number of words in refs[i].
