lingvo.core.inspect_utils module¶

Utilities to bind function signatures to params.

lingvo.core.inspect_utils.DefineParamsFromArgs(func, params, ignore=None)[source]¶

Defines params for each parameter of func.

This allows you to define the parameters necessary to call a function or instantiate a class without having to type the Define statements yourself. Default values for the function parameters will be copied into the params object as well.

To use this with a class constructor, use MyClass.__init__ as the func parameter.

Parameters

  • func – A callable. Parameters for this function will be defined in params.

  • params – A Params object to be updated. New parameters will be defined here.

  • ignore – A collection of parameter names in func to be ignored from defining resulting entries in params.

Extracts parameters from params that should be used to call func.

  • params – A hyperparams object containing arguments for func.

  • **kwargs – Argument/value pairs that should override params.
